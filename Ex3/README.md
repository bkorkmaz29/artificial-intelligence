# Tic-Tac-Toe  
A Tic-Tac-Toe game with an AI using minimax algortihm with alpha-beta pruning

## Dependincies
- Python3
- Numpy

## Installation
```sh
pip install pygame
```

```sh
pip install numpy
```
## How To Play
Run the main file using python
```sh
python3 main.py
```
To start second press 's'
To restart the game press 'r'
